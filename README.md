# Phùng Khánh Linh Artist Website

Diese Website ist mein erstes selbst erstelltes Webprojekt. Sie entstand, um die Grundlagen von HTML und CSS praktisch zu lernen und erste Erfahrungen mit dem Aufbau einer Website zu sammeln.

Inhaltlich beschäftigt sich die Website mit der vietnamesischen Sängerin Phùng Khánh Linh und präsentiert ausgewählte Informationen, Bilder und musikalische Inhalte.

## Projektziel

Das wichtigste Ziel des Projekts war es, die grundlegende Struktur einer Website zu verstehen und selbstständig umzusetzen.

Dabei habe ich gelernt, Inhalte mit HTML zu strukturieren und das Erscheinungsbild mit CSS zu gestalten. Ein besonderer Fokus lag auf Typografie, Farben, Abständen, Bildern und der Anordnung verschiedener Seitenelemente.

## Verwendete Technologien

Die Website wurde ausschließlich mit HTML5 und CSS3 erstellt.

Es wurden keine zusätzlichen Frameworks oder Programmiersprachen verwendet.

## Gestaltung

Die visuelle Gestaltung orientiert sich an der musikalischen Identität von Phùng Khánh Linh.

Farben, Bilder und Typografie wurden so ausgewählt, dass eine ruhige, persönliche und künstlerische Atmosphäre entsteht.

## Lernziele

Durch dieses Projekt konnte ich grundlegende Kenntnisse in folgenden Bereichen sammeln:

- Aufbau einer HTML-Seite
- Verwendung semantischer HTML-Elemente
- Gestaltung mit CSS
- Arbeit mit Farben, Schriftarten und Bildern
- Positionierung und Anordnung von Elementen
- Erste Erfahrungen mit responsivem Webdesign

## Projektstatus

Das Projekt wurde als erste Lernübung entwickelt und ist keine vollständig ausgearbeitete Künstlerwebsite.

Der Schwerpunkt liegt auf dem Lernprozess und auf der praktischen Anwendung der Grundlagen von HTML und CSS.

## Hinweis

Dieses Projekt wurde ausschließlich zu Studien- und Übungszwecken erstellt. Es besteht keine offizielle Verbindung zu Phùng Khánh Linh oder ihrem Management.
